# Client

**Module 2** -- PWA client connecting to the OpenClaw gateway via Matrix.

## Purpose

A Progressive Web App (PWA) that serves as the player-facing game client:
- Connects to the OpenClaw gateway through a Matrix homeserver
- Provides player-facing game controls and UI
- Supports offline mode via service worker

## Future

This package will contain:
- React-based PWA shell
- Matrix SDK integration (room management, messaging)
- Player game UI components
- Service worker for offline support

## Connects To

- OpenClaw Gateway (via Matrix protocol over HTTPS)
